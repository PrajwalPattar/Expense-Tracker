# 💰 Expense Tracker (MERN Stack)

A full-stack expense tracking web application built using the MERN stack.

---

## 🚀 Features

- User Authentication
- Add Income
- Add Expenses
- Budget Tracking
- Transaction History
- Dashboard Overview

---

## 🛠 Tech Stack

Frontend
- React.js
- CSS

Backend
- Node.js
- Express.js

Database
- MongoDB

---

├── client

## ⚙️ Installation

Clone the repository

git clone https://github.com/PrajwalPattar/Expense-Tracker.git

Go to project folder

cd Expense-Tracker

Install dependencies

npm install

Run the project

npm start
